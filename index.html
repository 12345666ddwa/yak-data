03数据字典

核心数据表字段定义，包含数据类型、约束条件、字段说明，共6表48字段。

TABLE 1 · population_records — 种群记录表（核心业务表）
#字段名类型长度约束说明
1idSERIAL—PK记录唯一标识，自增
2province_idINT—FK → regions.id省份，外键关联地理区域表
3yearSMALLINT—NOT NULL数据年份，范围1990-2030
4populationINT—NOT NULL存栏量，单位：头
5change_rateDECIMAL5,2—同比变化率，单位：%
6confidenceVARCHAR10—置信度：high/moderate/low
7source_idINT—FK → sources.id数据来源ID
8notesTEXT——备注说明
9created_atTIMESTAMP—DEFAULT NOW()记录创建时间
TABLE 2 · geographic_regions
字段类型说明
idSERIALPK，区域唯一标识
provinceVARCHAR(30)省份名称，UNIQUE
elevationVARCHAR(20)海拔范围，如"3,200-4,800"
climateVARCHAR(30)气候类型：高原寒温/高山亚寒
grasslandVARCHAR(30)草场类型：高寒草甸/山地草原
area_km2INT牧区面积(km²)
carryingINT理论载畜量(万头)
TABLE 3 · breed_standards
字段类型说明
idSERIALPK
breed_nameVARCHAR(30)品种名称，UNIQUE
pctDECIMAL(4,1)种群占比(%)
wt_maleINT公牛体重(kg)
wt_femaleINT母牛体重(kg)
milk_fatDECIMAL(3,1)乳脂率(%)
slaughterDECIMAL(4,1)屠宰率(%)
实体关系说明

04数据查询

基于核心表的多维度交叉查询与分析。

📋 数据查询示例
SELECT p.province, r.elevation, p.year, p.population
FROM population_records p
JOIN geographic_regions r ON p.province_id = r.id
WHERE p.year = 2023
AND r.elevation LIKE '%4000%'
ORDER BY p.population DESC;
